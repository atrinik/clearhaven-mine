The different Frames and their Purpose

Icon Toolbar:
Contains shortcuts for certain commands.

Default-Archpanel:
This panel contains all the available default arches. You can select from the list and then eventually insert them into your map.

Arch Info:
This small window contains some basic info about the selected arch from the default-archpanel. Personally I prefer to squeeze this to zero height, in order to have more space for the default-archpanel itself.

Map View:
The Map View shows the currently open and active map. Here you can select, insert and delete objects.

Map-Archpanel:
This panel contains a list of all objects on the currently active (/selected) map square. Here you can select/insert/delete single objects, similar to how it works in the map view. Additionally you can change the stacking order of the objects by pressing the up/down arrows below the panel.

Object Attribute Window:
Here you have a small interface for the most basic attributes of the currently selected object. You can change it's name, face and message-text. Don't forget to click "Apply" to apply the new settings.

Archtext Window:
This window contains the arch-attributes of the currently selected object. You should know what you're doing when modifying these. See Manipulating Objects.
Again, don't forget to hit "Apply".

Status Bar:
On the left side, the status bar displays the map-coordinates of your mouse pointer as you move it across the map view. On the right side, the current memory usage is displayed.
